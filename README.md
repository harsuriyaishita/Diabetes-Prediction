# Diabetes Prediction Using Machine Learning

This project implements a **predictive model for diabetes** using clinical and lifestyle data. The aim is to help healthcare providers **identify individuals at risk** of developing diabetes and provide **early intervention recommendations**.  

The dataset includes features such as:  
- **Age, Gender**  
- **BMI (Body Mass Index)**  
- **HbA1c Level**  
- **Blood Glucose Level**  
- **Hypertension & Heart Disease**  
- **Smoking History**  

The project workflow includes:  
1. **Data Preprocessing:** Handling missing values, normalization, and encoding categorical features.  
2. **Exploratory Data Analysis (EDA):** Visualizing feature distributions and correlations.  
3. **Model Training:** Implementation of various machine learning models like Logistic Regression, Decision Trees, Random Forest, and XGBoost.  
4. **Model Evaluation:** Using metrics such as Accuracy, Precision, Recall, F1-score, and Confusion Matrix.  
5. **Power BI Dashboard:** Visual representation of predictions, risk analysis, and model performance for stakeholders.  

**Technologies Used:**  
- Python (Pandas, NumPy, Scikit-learn)  
- Power BI for dashboard visualization  
- Colab

**Objective:**  
To provide a **user-friendly, data-driven tool** for predicting diabetes risk, enabling **early diagnosis and preventive care**.
